**Documentation for the is_prime function:**

**Function:** is_prime

**Description:**
Checks if a given number is prime.

**Parameters:**

* number: The number to check.

**Return Value:**

* True if the number is prime, False otherwise.

**Example:**

```python
>>> is_prime(7)
True
>>> is_prime(10)
False
```

**Additional Notes:**

* A prime number is a positive integer greater than 1 that is not divisible by any other number except 1 and itself.
* The function uses a basic algorithm to check primality, which has a time complexity of O(√n), where n is the input number.

**Complexity:**
Time complexity: O(√n), where n is the input number.
